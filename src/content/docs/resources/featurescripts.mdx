Onshape has its own scripting language to create custom features to automate aspects of design called [Featurescript](https://www.onshape.com/features/custom-features). There are a lot of featurescripts (what we call the custom features) that make designing for FRC much easier.

<Aside type="tip">
The title of each featurescript or group of featurescripts links to its respective document. Right click the title and click "copy link address" for installing it to your profile using the process described on [the featurescript help page](/resources/featurescript-help/).
</Aside>

## Recommended
These featurescripts are must haves to enhance your Onshape workflow. Many of these are developed by members of the FRC community and well maintained.

<CardGrid>
<CustomCard title="Origin Cube Featurescript" subTitle="" href="https://cad.onshape.com/documents/321c197a842fc5f1a29e6621/w/fc3cdd5ca7edcd93e02f13cc/e/c66db2b954da7f29d438195a">
<hr/>
<ContentFigure src="./img/featurescripts/origin-cube.webp" alt="Origin Cube" width="80%" />

- Used to create an origin reference for assembly mating
- Optionally import design constants like bolt hole sizes, heat set insert hole sizes, etc
- Optionally import functions for calculating gear OD, pulley/sprocket pitch & outer diameters, and calculate belt/chain c-c distances
- See [Assembly Best Practices](/best-practices/assembly-setup/#the-origin-cube) for more information on usage

<Aside type="video" title="Video Example" collapse>
<ContentFigure src="https://www.youtube.com/watch?v=3m4QLn2aSXQ" />
</Aside>

</CustomCard>
<CustomCard title="Belt & Chain Gen" subTitle="" href="https://cad.onshape.com/documents/53c0b14cad92676c14e04e97/w/1271c254ccb0a79563210195/e/7394c4a86d8d6c35c9a12041">

<hr/>
<ContentFigure src="./img/featurescripts/belt-chain-gen.webp" alt="Belt & Chain Gen" width="80%" />

- Used in conjunction with Origin Cube FRC Functions to create highly customizable belts and chains
- Highly parametric - pulley/sprocket tooth count is automatically calculated from sketch pitch diameter.
- Create multi pulley, double sided belt, and multi sprocket transmissions with options for both toothed and round idlers

<Aside type="video" title="Video Example" collapse>
<ContentFigure src="https://www.youtube.com/watch?v=n_kc2RV-qgI" />
</Aside>

</CustomCard>
<CustomCard title="Robot Shaft" subTitle="(Alex's Featurescripts)" href="https://cad.onshape.com/documents/9cffa92db8b62219498f89af/w/06b332ccabc9d2e0aa0abf88/e/2e6e4b559832eeff8391e933">

<hr/>
<ContentFigure src="./img/featurescripts/robot-shaft.webp" alt="Robot Shaft" width="80%" />

- Generates a shaft, with options for hex, rounded hex (13.75mm OD), MAXSpline, etc.
- Options for retention type on both ends
- Can be generated in-place and with offsets

</CustomCard>
<CustomCard title="Robot Spacer" subTitle="(Alex's Featurescripts)" href="https://cad.onshape.com/documents/9cffa92db8b62219498f89af/w/06b332ccabc9d2e0aa0abf88/e/2e6e4b559832eeff8391e933">

<hr/>
<ContentFigure src="./img/featurescripts/robot-spacer.webp" alt="Robot Spacer" width="80%" />

- Generates a spacer, with options for round, 3/8" hex, 1/2" hex, MAXSpline, etc.
- Can be generated in-place and with offsets

</CustomCard>
<CustomCard title="Extrude Individual" subTitle="(Julia's Featurescripts)" href="https://cad.onshape.com/documents/95c00401c440b44ad8799ef5/w/1f1ebce01a3b8eb6fa102975/e/7f62400703032fab9a7a3320">

<ContentFigure src="./img/featurescripts/extrude-individual.webp" alt="Extrude Individual" width="80%" />

- Can extrude multiple regions as different parts with one feature, even if regions are adjacent
- Commonly used with tube converter to make punched aluminum tubes from a top view sketch

<ContentFigure src="./img/featurescripts/tube-converter.webp" alt="Tube Converter" width="80%" />

## Other
Below is a list of other featurescripts that can be useful but are more niche.

### Structure

<CardGrid>
<CustomCard title="Gusset Generator" subTitle="(Julia's Featurescripts)" href="https://cad.onshape.com/documents/95c00401c440b44ad8799ef5/w/1f1ebce01a3b8eb6fa102975/e/7f62400703032fab9a7a3320">

- Generate a gusset from hole selections
- **Warning**: Not a very parametric featurescript and causes lag, but can be useful in a pinch

</CustomCard>
<CustomCard title="Robot Bearing Hat" subTitle="(Alex's Featurescripts)" href="https://cad.onshape.com/documents/9cffa92db8b62219498f89af/w/06b332ccabc9d2e0aa0abf88/e/2e6e4b559832eeff8391e933">

- Creates a custom bearing hat given a set of mounting locations and a bearing hole
- Can create tapped holes

</CustomCard>
<CustomCard title="Spacer Generator" subTitle="(Julia's Featurescripts)" href="https://cad.onshape.com/documents/95c00401c440b44ad8799ef5/w/1f1ebce01a3b8eb6fa102975/e/7f62400703032fab9a7a3320">

- Generates a spacer, with options for round, 3/8" hex, and 1/2" hex

<Aside type="video" title="Video Example" collapse>
<ContentFigure src="https://www.youtube.com/watch?v=Wnz442Dv-uw" />
</Aside>

</CustomCard>
<CustomCard title="Electronics Mounting" subTitle="(Julia's Featurescripts)" href="https://cad.onshape.com/documents/95c00401c440b44ad8799ef5/w/1f1ebce01a3b8eb6fa102975/e/7f62400703032fab9a7a3320">

- Generates hole patterns for different electronic components using center points and mate connectors on sketches

</CustomCard>
<CustomCard title="MechSketch" subTitle="(TLamp's Featurescripts)" href="https://cad.onshape.com/documents/f46510f08cc311a34733cac2/v/f7f3a1c996c58466f6b88a37/e/8e9fbf1ac2f0980e4bc34951">

- Incredibly useful, generates sketch profiles for a variety of things
- Profiles for COTS parts like Maxspline, TTB inserts, maxtube, motors, and the versa patern
- Profiles and holes for all electronics
- Profile generator for pulleys and a spur gears

</CustomCard>
<CustomCard title="Robot Spline Profile" subTitle="(Alex's Featurescripts)" href="https://cad.onshape.com/documents/9cffa92db8b62219498f89af/w/06b332ccabc9d2e0aa0abf88/e/2e6e4b559832eeff8391e933">

- Creates an inside or outside profile/extrusion of MAXSpline or SplineXL
- Has all the standard extrude parameters

</CustomCard>
</CardGrid>

### Power Transmissions

<CardGrid>
<CustomCard title="Robot Pulley" subTitle="(Alex's Featurescripts)" href="https://cad.onshape.com/documents/9cffa92db8b62219498f89af/w/06b332ccabc9d2e0aa0abf88/e/2e6e4b559832eeff8391e933">

<CustomCard title="Belt & Pulley Generation" subTitle="(2471 Featurescripts)" href="https://cad.onshape.com/documents/028ca8fb10baf53e1f6fce96/v/821c8b51ed0953526b51926e/e/a8b9e45297aac9f5688c871d">

</CustomCard>
<CustomCard title="Timing Belt Pulley Generator" subTitle="" href="https://cad.onshape.com/documents/55ae3138e0cd7c325239208b/v/bfe7a53a6752a35e56338377/e/b08c0071a21e9ab74baaf916">

</CustomCard>
<CustomCard title="Sprocket" subTitle="(Abhi's Features & Configs)" href="https://cad.onshape.com/documents/96d32fd4fe5341563a402ecd/v/042060f84c30390d82d806f8/e/d7f04278e0325b6d4426ca34?jumpToIndex=2635&showReturnToWorkspaceLink=true">

</CustomCard>
<CustomCard title="Polybelt Generator" subTitle="(2471 Featurescripts)" href="https://cad.onshape.com/documents/028ca8fb10baf53e1f6fce96/v/821c8b51ed0953526b51926e/e/a8b9e45297aac9f5688c871d">

### Manufacturing

<CardGrid>
<CustomCard title="Auto Layout" subTitle="" href="https://cad.onshape.com/documents/576e01dbe4b0cc2e7f46a55d/v/f697250aa149b6846173f4b1/e/887d6e2324589bfd2058c3e1?jumpToIndex=2346">

- Given some derived plates, either transformed from an in-context assembly (to maintain part count) or derived from a part studio, it will automatically lay them out into sheets of a given size, sorted by thickness. This is used to prepare a file to route or waterjet parts out of material with a machine.

</CustomCard>
<CustomCard title="Measure Cut List" subTitle="" href="https://cad.onshape.com/documents/395304920b30c3b382fd312d/w/86481637ad1e9da01b0e5d60/e/e19230df69a29e1fc2259b9d?renderMode=0&tangentEdgeStyle=2&uiState=6642611f3aed1a692ef01f41">
